# SSG Website - Student Sponsor Gateway

A premium NIL (Name, Image, Likeness) platform connecting college athletes with brand partnerships. Built with React, TypeScript, Framer Motion, and Tailwind CSS, featuring Tykoon-inspired animations and design.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build

## Deployment

Build the project using `npm run build` and deploy the `dist` folder to your hosting provider.
